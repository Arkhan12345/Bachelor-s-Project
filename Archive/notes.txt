Research question:
How effectively can LLMs automatically annotate Independent Component Analysis derived transcriptional components by linking them to relevant biological themes in scientific literature?

Hypothesis:
AI-powered literature mining can effectively reveal the biological functions underlying transcriptional components (TCs) derived from cancer transcriptomes.


Linking files:
Independent_components are the underlying statistically independent transcriptional components.
Rows of this file are the genes. The description of the genes is present in genomic_mapping file.
The geneset enrichment analysis of the components are saved inside the file gsea_matrix.csv.
Mixing_matrix shows the activity scores of those components in the samples.
The details about the samples are provided in the file sample_annotations.txt.
